# 04 報酬ループ — 「やればやるほど資材」設計

## 設計思想

学習量（正解数・収穫数）が、**その後ずっと効く資産** に変わる感覚を作る。
「今日の苦労は明日の自分が楽になる」を体感させる。

## ループ階層

3段階のループを同時に走らせる：

```
[即時]  正解 → GP獲得・コンボ・米粒小ドロップ            (秒)
[中期]  収穫 → 大量米粒・品種コレクト・アチーブ進捗      (分〜時間)
[長期]  恒久アップグレード → 田んぼ性能恒久UP            (日〜週)
```

現状は即時・中期は揃っているので、**長期ループの強化**にフォーカスする。

## 新規: 恒久アップグレード（Permanent Upgrades）

### 1. 田んぼ別アップグレード

各田んぼに以下のスロットを設ける。米粒で強化、永続的に効果。

| アップグレード | 効果 | レベル | 価格カーブ |
|---|---|---|---|
| 肥沃度 (soil_fertility) | 獲得GP +5% / Lv | 0-10 | 200×1.6^Lv |
| 水路 (irrigation) | 干ばつ・冷夏耐性 +10% / Lv | 0-5 | 400×1.8^Lv |
| 防風林 (windbreak) | 台風・強風耐性 +15% / Lv | 0-5 | 600×1.8^Lv |
| 害虫網 (pest_net) | 病害虫イベント発生率 -10% / Lv | 0-5 | 500×1.8^Lv |
| 苗床品質 (nursery) | 田植え時にスタートGP+10/Lv | 0-5 | 300×1.7^Lv |

合計コスト目安: 全Lv MAXで約30,000米粒。長期目標として機能。

### 2. プレイヤー全体アップグレード（農具・知識）

田んぼ横断的に効くもの。ホーム画面 or 設定タブから購入。

| アップグレード | 効果 | レベル | 価格 |
|---|---|---|---|
| 鎌の品質 (sickle) | 収穫量 +5% / Lv | 0-10 | 500×1.6^Lv |
| 農学知識 (knowledge) | クイズ難易度ボーナス +0.1×Lv | 0-5 | 1000×2.0^Lv |
| インベントリ拡張 (inventory) | アイテムスタック上限 +5/Lv | 0-5 | 300×1.5^Lv |
| 倉庫 (warehouse) | 翌日持ち越し米粒上限 +1000/Lv | 0-10 | 800×1.7^Lv |
| 種苗カタログ (catalog) | ショップに新品種枠 +1/Lv | 0-5 | 1500×2.0^Lv |

### 3. 累計実績ボーナス（Prestige風）

通算収穫数や累計問題数で恒久的に解放される効果。

| 条件 | 永続効果 |
|---|---|
| 累計100問正解 | 米粒獲得+5% |
| 累計500問正解 | コンボ最大値解放 (20→30) |
| 累計1000問正解 | 全GP獲得+10% |
| 累計収穫10回 | 田植え時に肥料1自動付与 |
| 累計収穫50回 | 全イベント発生率-15% |
| 累計収穫100回 | 「伝説の農夫」称号＋特別品種 |

## 報酬感覚を強化する細部

### a. ドロップアニメーション
収穫時、画面中央にコイン (🍚×N) が舞うアニメ。1秒。

### b. 数字の急上昇エフェクト
米粒残高がカウンターアニメで増えていく演出（200ms）。

### c. 「もうちょっと」表示
ショップで「あと150米粒で買える」のような近接表示。次の正解で米粒が増えるたびに更新。

### d. 通算実績の可視化
ホーム画面のステータスを大きく見せる（累計問題数・収穫数・所有品種数を3桁数字でドンと）。

### e. 価格カーブの設計原則
- アップグレードはレベル毎に1.5〜2.0倍に増価
- 序盤(Lv1-2)は1日で買える、終盤(Lv5+)は1週間がかりで買える
- 「次のレベルが買えるとき」のタイミングが、平均プレイ時間と噛み合うようバランス

## ショップ拡張

現状のショップに「アップグレード」タブを追加：

```
🏪 ショップ
├─ 🧰 道具・肥料   (現状)
├─ 🌾 品種        (現状)
├─ 🛠️ アップグレード (新規) ← 田んぼ別 / 全体別の2セクション
└─ 🎁 装飾         (将来)
```

## バランス調整のためのテレメトリ（軽量）

学習用のため、外部送信なし。ローカルにメトリクスを記録：

- 1セッションあたりの問題数中央値
- 1日あたりの米粒獲得・消費の比
- アップグレード購入率
- どのアイテムが最も使われているか

設定画面に「統計を見る」ボタンを置き、自分で確認できるようにする。

## やらないこと（ループ深さの線引き）

将来検討するが、今は実装しない：

- 加工施設（精米所・酒蔵）
- 市場価格の変動
- NPC農家との取引
- 季節サイクル（春夏秋冬）
- マップ拡張（隣の村）

これらは v2.0 以降の話題。v1 は「学習→田んぼ→アップグレード」の3層を磨く。
